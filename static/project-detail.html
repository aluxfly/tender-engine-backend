项目概述

本项目为北京市朝阳区政府办公楼装修工程，总建筑面积约 5000 平方米。装修范围包括办公区域、会议室、接待室、休息区、卫生间等公共区域。 要求采用环保材料，符合绿色建筑标准，工期 90 日历天。投标人需具备建筑装修装饰工程专业承包二级及以上资质。

招标范围

  • 室内墙面、地面、天花装修工程
  • 电气照明系统安装工程
  • 给排水改造工程
  • 空调通风系统工程
  • 智能化系统集成工程
  • 消防设施改造工程

资质要求

  • 建筑装修装饰工程专业承包二级及以上资质
  • 安全生产许可证在有效期内
  • 近三年类似项目业绩不少于 3 个
  • 项目经理具备二级及以上注册建造师资格
  • 具有良好的商业信誉和财务状况

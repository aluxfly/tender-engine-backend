## 新增接口

### 1. 关键词筛选

**URL：** `GET /api/projects/filter?keywords=物联网卡，布控球`

**实现：**
```python
@app.get("/api/projects/filter")
async def filter_projects(keywords: str = ""):
    keyword_list = keywords.split(",")
    projects = db.query(Project).filter(
        or_(*[Project.name.contains(kw) for kw in keyword_list])
    ).all()
    return projects
```

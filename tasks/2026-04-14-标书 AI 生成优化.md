## 功能需求

### 1. 行业模板库

**物联网卡投标模板：**
- 技术方案（物联网卡技术参数、网络覆盖、资费方案）
- 商务方案（供货周期、售后服务、质保期）
- 资质要求（通信资质、ISO 认证等）

**布控球投标模板：**
- 技术方案（布控球参数、视频质量、传输方式）
- 商务方案（安装调试、培训、售后）
- 资质要求（安防资质、3C 认证等）

### 2. 技术参数库

**物联网卡参数：**
```json
{
  "网络制式": ["2G", "3G", "4G", "5G", "NB-IoT"],
  "运营商": ["移动", "联通", "电信"],
  "流量套餐": ["1GB/月", "5GB/月", "10GB/月", "不限量"],
  "卡类型": ["插拔卡", "贴片卡", "eSIM"],
  "工作温度": ["-40°C~85°C", "-20°C~70°C"],
  "接口类型": ["Mini SIM", "Micro SIM", "Nano SIM"]
}
```

**布控球参数：**
```json
{
  "摄像头像素": ["200 万", "400 万", "800 万"],
  "视频编码": ["H.264", "H.265"],
  "传输方式": ["4G", "5G", "WiFi", "有线"],
  "电池续航": ["4 小时", "8 小时", "12 小时"],
  "防护等级": ["IP65", "IP66", "IP67"],
  "夜视距离": ["30 米", "50 米", "100 米"]
}
```

### 3. 资质要求库

**通信行业资质：**
- 增值电信业务经营许可证
- ISO9001 质量管理体系认证
- ISO14001 环境管理体系认证
- 3C 认证（强制性产品认证）

**安防行业资质：**
- 安防工程企业设计施工维护能力证书
- ISO9001 质量管理体系认证
- 3C 认证
- 公安部检测报告

### 4. 案例库

**物联网卡案例：**
- 某市智能水表项目（物联网卡供应）
- 某物流公司车联网项目
- 某共享经济项目（共享单车/充电宝）

**布控球案例：**
- 某市公安局移动布控项目
- 某高速公路临时监控项目
- 某大型活动安保布控项目

### 5. AI 生成优化

**输入：**
- 招标文件（PDF/Word）
- 项目基本信息

**输出：**
- 技术方案（针对物联网卡/布控球优化）
- 商务方案（标准模板 + 自定义）
- 资质证明（自动匹配）
- 案例推荐（类似项目）

**优化点：**
- 行业术语自动填充
- 技术参数智能匹配
- 资质要求自动检查
- 废标项自动预警

## 技术实现

### 后端（dev-backend）

**API 接口：** `POST /api/bid/generate`

**请求参数：**
```json
{
  "projectType": "物联网卡" | "布控球",
  "projectInfo": {
    "name": "项目名称",
    "budget": 500000,
    "company": "甲方名称",
    "deadline": "2026-05-01"
  },
  "tenderDocument": "招标文件内容（可选）",
  "customParams": {
    // 自定义参数
  }
}
```

**返回格式：**
```json
{
  "status": "success",
  "bidDocument": {
    "technicalProposal": "技术方案内容",
    "businessProposal": "商务方案内容",
    "qualifications": ["资质 1", "资质 2"],
    "cases": ["案例 1", "案例 2"]
  },
  "warnings": ["废标项预警"],
  "downloadUrl": "下载链接"
}
```

### 前端（dev-frontend）

**页面：** `/bid/generate`

**功能：**
- 选择项目类型（物联网卡/布控球）
- 填写项目信息
- 上传招标文件（可选）
- 预览生成的标书
- 编辑和导出

### 数据（pm）

**需要整理：**
- 物联网卡技术参数表
- 布控球技术参数表
- 行业资质要求清单
- 典型成功案例（各 5-10 个）
